# Одномерный анализ

Предположим, что у нас есть функция 
$$
f:R \to R
$$


Будемм считать, что она непрерывна и дифференцируема 


Определим понятние производной

$$
f'(x_0) =\frac{f(x_0 + \Delta x) - f(x_0)}{\Delta x}
$$
